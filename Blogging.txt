Today I want to understand better how the C++ language works, so I'll try to solve a LeetCode question using C++.
And since I'm a complete beginner with the syntax, I'll start with a VERY simple challenge: transforming an int into a string.

So let's say we have an int variable with the value 10, and we want the result to be the string "10". In Python, this would be as simple as `str(10)`, but I have no idea if the same can be done in C++.

I've already had a TON of issues. Apparently, you need to `#include <string>` just to be able to use strings in your code.
After a few searches, I found the `std::to_string` function from the `<string>` library, and this does exactly what I need it to do.

So while it's not as simple as in Python, it's still pretty simple. As for the solution, I have:

```cpp
#include <string>

std::string number_to_string(int num) {
    return std::to_string(num);
}
```

I'm quite happy with the result, and while I don't know if this is "good code," it solved the issue in just a single line.
